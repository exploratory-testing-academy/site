Exploratory Testing Academy

Exploratory testing, coined 35 years ago by Cem Kaner, is a multidisciplinary and learning-centric approach to testing. The core of it is learning, agency, results, opportunity cost, systems thinking. It includes creating systems to test systems we know as learning-focused application of test automation.

All good testing is exploratory to an extent, but not all testing is exploratory testing.

Exploratory Testing is the craft of learning well while testing software so that noticing important information about the software is inevitable.
